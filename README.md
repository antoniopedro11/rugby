# 🏉 Rugby Scoreboard App

Um aplicativo de placar visual, interativo e moderno para gerenciar partidas de Rugby.

## Funcionalidades Incluídas
- **Pontuação Rápida:** Botões de fácil acesso para *Tries* (5 pts), *Conversões* (2 pts) e *Pênaltis/Drop Goals* (3 pts).
- **Cronômetro:** Relógio de jogo que conta até 80 minutos com funções de Iniciar (▶), Pausar (⏸) e Zerar (⏹).
- **Cartões:** Sistema para adicionar Cartões Amarelos (🟨) e Vermelhos (🟥) para cada time separadamente.
- **Correção de Pontos:** Botão de correção de acidentes (-1 ponto) em cada lado.
- **Customização de Times:** Clique em cima de "Local" ou "Visitante" para digitar o nome real dos times da partida.
- **Interface Web:** Totalmente feito com HTML, CSS e JavaScript Vanilla. Interações rápidas com animações suaves e sem dependências extras.
- *(Bônus)* **Placar de Linha de Comando:** Vem com um script Ruby opcional (`app.rb`) para quem prefere rodar via terminal.

## Como Executar a Versão Visual

Basta dar dois cliques no arquivo `index.html` ou abri-lo pelo seu navegador favorito (Chrome, Safari, Firefox, etc). Nenhum servidor local é necessário e funciona 100% offline.
